Bienvenido a Bracotes Resources — una colección curada de recursos web para desarrolladores: artículos, librerías, cursos, utilidades y herramientas útiles organizadas por categorías.

Propósito
- Reunir enlaces y recursos de calidad para desarrolladores front-end, back-end, devops, diseño y más.
- Facilitar la colaboración: cualquiera puede proponer y añadir recursos siguiendo la guía de contribución.

Cómo navegar
- Estructura principal (ejemplo):
  - /frontend
  - /backend
  - /devops
  - /design
  - /tools
  - Archivos JSON que contienen arrays de recursos (ej. `data/resources.json`, `frontend/resources.json`, etc.)
- Usa el archivo `SUMMARY.md` (índice) para ver categorías y contenidos.
- En este repo el flujo de contribución consiste en añadir un objeto JSON a los arrays existentes en los archivos de recursos (ver `CONTRIBUTING.md` para el formato y la ubicación). No se usa una plantilla Markdown por recurso.

Ejemplo de recurso (resumen)
- Título: Awesome CSS Tricks
- Tip: article · frontend · CSS
- Link: https://ejemplo.com/awesome-css

Cómo contribuir
1. Lee `CONTRIBUTING.md` y localiza el archivo JSON donde añadir el recurso.
2. Crea una rama con tu aporte.
3. Añade el objeto JSON al array correspondiente y actualiza `SUMMARY.md` si aplica.
4. Abre un PR con descripción y cambios (usa el template de PR).
5. Las contribuciones serán revisadas y validadas automáticamente por las Actions del repo.

Archivos importantes
- README.md — este archivo.
- SUMMARY.md — índice y navegación.
- CONTRIBUTING.md — guía y formato JSON para nuevas aportaciones.
- .github/ISSUE_TEMPLATE/ — plantilla para issues.
- .github/PULL_REQUEST_TEMPLATE.md — plantilla para PRs.
- CODE_OF_CONDUCT.md — normas de convivencia.
- LICENSE — licencia MIT.
