# Código de Conducta

Nos basamos en el [Contributor Covenant](https://www.contributor-covenant.org/). Resumen y expectativas:

- Sé respetuoso y considerado con todas las personas.
- No toleramos comportamientos discriminatorios, de acoso o abusivos.
- Usa un lenguaje profesional y evita ataques personales.
- Si experimentas o ves comportamientos inapropiados, por favor abre un issue o contacta a los mantenedores.

Para más detalles y el texto completo del Covenant, visita:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Contacto para reportes: @BracoZS (o abrir un issue privado si procede).
